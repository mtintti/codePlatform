📝 How to Build
Follow these steps to build the packages:

Open a terminal (Command Prompt.
Clone the repository:
git clone https://github.com/mtintti/codePlatform.git
Navigate to the project directory:
cd codePlatform
Install dependencies:
npm install
Start the project:
npm run dev
